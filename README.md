🎯 What is Fluency?
Fluency is a galaxy-themed impromptu speaking trainer that gives you random topics + a timer to practice thinking and speaking on your feet.

Perfect for:

🎤 Public speaking practice

💼 Interview preparation

🗣️ Toastmasters training

📝 Exam prep (IELTS/TOEFL)

✨ Features
Feature	Description
🎲	Random topics from your CSV files
⏱️	Prep timer + Speak timer (15s–5min)
📂	Unlimited topics 
⌨️	Keyboard shortcuts (Space, N)
🔊	Sound alerts (toggle on/off)
🌌	Animated galaxy theme

🎮 How to Use
Select category (General / Professional)

Set timers (Prep: 30s, Speak: 2min)

Click "NOVA TOPIC" for a random prompt

Press SPACE or click "INITIATE"

Prepare during prep time

Speak during speak time

Complete + get credit!

Keyboard Shortcuts:
Space → Start / Pause timer

N → New random topic
