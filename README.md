# 🌌 Fluency Galaxy  
### *Impromptu Speaking Practice | Speak Without Limits*

---

## 🚀 Overview

**Fluency Galaxy** is a galaxy-themed impromptu speaking trainer designed to help users improve confidence, spontaneity, and communication skills through random speaking prompts and timed practice sessions.

Whether you're preparing for interviews, public speaking competitions, language proficiency exams, or simply trying to become a better communicator — Fluency Galaxy helps you think fast and speak clearly.

---

## 🎯 What is Fluency Galaxy?

Fluency Galaxy gives you:

- 🎲 Random impromptu speaking topics
- ⏱️ Custom prep + speaking timers
- 🌌 Immersive animated galaxy UI
- 🔊 Audio alerts for transitions
- ⌨️ Keyboard shortcuts for smooth practice

The goal is simple:

> **Train your mind to think quickly and speak confidently without preparation.**
